3. tạo project (index.html)
4. thực hành lệnh git
 - cho git theo dõi: git init
 - xem tình trạng của local repo: git status
 - add các sự thay đổi: git add . (index.html, *)
 - hình thành một version: git commit -m "add file index.html"
